The To-Do list consists os operations such as adding a new task,marking a new task as complete,deleting a task and searching for a new task.
1.To add a new task,type the task adjacent to the 'Add task' button and click on the 'Add task' button to the task to be added.
2.To mark a task as complete tick the checkbox next to the task that is to be marked as complete.
3.To delete a task, click on the delete button next to the task which is to be deleted.
4.To search for a task,Type the name of the task in the box below the add task.
